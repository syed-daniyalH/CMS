# Techionik CMS Dashboard

## Overview

Techionik CMS Dashboard is a Next.js admin panel for managing website content and business-facing data from a single interface. The project combines CMS workflows with CRM-style modules for pages, services, industries, case studies, careers, clients, reports, and operational records.

## Key Features

- Content section and page management for website publishing workflows
- Blog, category, and sub-category administration
- Service, sub-service, technology, and industry management
- Career, hiring, and client management screens
- Case study and marketing content administration
- Role-based access control and permission-aware navigation
- Reporting and inventory-operation modules for business workflows
- Media upload, attachment, and form helper components

## Tech Stack

- Next.js
- TypeScript
- Material UI
- Apollo Client
- CASL permission management

## Business Use Case

This repository is best suited for companies that need a web-based admin dashboard to manage marketing content, CRM-style data, structured business records, and reporting workflows from a centralized interface.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Review the included environment examples:

- `.env.example`
- `.env.local.backend.example`
- `.env.target.example`

3. Start the local development server:

```bash
npm run dev
```

## Project Status

Active admin frontend workspace. The repository contains substantial business modules and UI workflows, but the backend and deployment environment should be configured separately.
