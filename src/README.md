# Frontend Source Map

This source tree is organized to separate the app framework, active features, and example/template code.

## Main Folders

- `core` - Shared app foundation, utilities, theming, and internal UI primitives
- `components` - Reusable custom UI pieces used across the app
- `configs` - App configuration values
- `context` - React context providers and auth state
- `examples` - Template/demo pages and views kept for reference
- `graphql` - GraphQL helpers and query definitions
- `hooks` - Shared React hooks
- `icons` - Icon generation/bundle tooling
- `layouts` - Layout shells and chrome
- `mock-db` - Mock data and fake auth data
- `navigation` - Menu and nav definitions
- `pages` - Next.js route entry points
- `previews` - Print/PDF preview components
- `store` - Redux state and slices
- `types` - Shared TypeScript types
- `views` - Production feature views and page sections
